# CSE-PROJECT
PROJECT TITLE: SNAKE AND LADDERRS GAME
OVERVIEW OF THE PROJECT :
                         This is a simple classic board game Snake and Ladders implemented in Python.
The game simulates the movement of two players on a 100-block board where ladders help the player move ahead while snakes pull the player back.
The project focuses on basic Python concepts such as loops, functions, random number generation, and conditional statements.

FEATURES :
Two-players gameplay

 Dice roll using random module

Snakes and ladders positions predefined

Prevents player from crossing 100

Exact 100 needed to win

Clean turn-based output

Fully console-based game


TECHNOLOGIES / TOOLS USED :
                            Python 3.x
                            Modules: random

INSTRUCTIONS FOR TESTING:

1. Run the program in any Python IDE/terminal.
2. Two players take turns automatically.
3. The dice rolls randomly every turn.
4. When a player reaches a cell with,

Ladder → Player climbs up
Snake → Player slides down

5.  When any player reaches exactly 100 then game will end.
